    # STAY MOOD

감성 스테이 및 쉼을 주제로 한 반응형 웹사이트 메인 페이지입니다. Swiper 라이브러리를 활용한 메인 슬라이드와 스테이 소개, 서비스, 리뷰 섹션으로 구성되어 있습니다.

## 🛠️ 사용 기술
* **HTML5** : 시맨틱 태그를 활용한 웹 표준 구조 설계
* **CSS3** : Flexbox 레이아웃 모델을 활용한 정렬 및 스타일링
* **Swiper JS (v12)** : 터치 슬라이드 내비게이션 구현

## 🖥️ 주요 화면 구성 및 기능

### 1. 헤더 (Header)
* 상단 고정(`position: fixed`) 레이아웃 적용
* 로고, 글로벌 내비게이션(GNB), 예약하기 버튼이 가로 방향(`display: flex`)으로 정렬

### 2. 메인 슬라이드 (Main Slide)
* Swiper JS 플러그인을 활용한 슬라이드 구현
* 슬라이드 하단에 페이지 번호(Pagination) 노출
* 각 슬라이드 이미지 중앙에 메인 텍스트 영역(`position: absolute`) 배치

### 3. 어바웃 섹션 (About Section)
* 가상 요소(`::before`, `::after`)를 사용하여 배경 화면을 좌우 50% 비율의 이중 이미지로 분할 처리
* 우측 레이어 상단에 텍스트를 배치하고 중앙 정렬 처리

### 4. 서비스 섹션 (Service Section)
* '머무름을 위한 서비스'를 안내하는 4개의 아이콘 카드 배치
* `gap: 150px` 수치를 적용하여 각 서비스 항목을 가로로 여유 있게 나열

### 5. 리뷰 섹션 (Review Section)
* 스테이 방문객들의 후기를 담은 3열 카드 레이아웃
* 각 마우스 오버(`:hover`) 시 후기 이미지가 부드럽게 확대(`transform: scale(1.1)`)되는 인터랙션 효과 적용
