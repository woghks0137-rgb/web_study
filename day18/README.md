# 📝 2026.06.01 학습 내용 정리

## 1. 프로젝트 개요
* **서브메뉴가 있는 네비게이션**: `position` 속성을 활용한 2차원 드롭다운 메뉴(GNB) 레이아웃 구현
* **국경없는의사회 소개 페이지**: 시맨틱 태그 활용, 유튜브 영상(`iframe`) 임베드, 카드 레이아웃 정렬을 포함한 상세 웹페이지 마크업 및 스타일링

---

## 2. 주요 구현 기능 및 기술 요약

### 📂 서브메뉴(Dropdown) 네비게이션
* **레이아웃 구조**: 상단 헤더 내 1차 메뉴(`gnb`)와 각 항목 하위의 2차 메뉴(`depth2`)를 중첩 목록(`ul > li > ul`) 구조로 마크업
* **서브메뉴 숨김 및 노출**: 
  * `.depth2`에 `display: none;`을 기본값으로 주고, 부모 요소 hover 시(`.gnb>li:hover .depth2`) `display: block;`으로 전환되도록 처리
  * 마우스가 서브메뉴 영역으로 이동해도 메뉴가 사라지지 않도록 부모인 `li` 요소에 hover 이벤트를 바인딩
* **정적 위치 제어**: 부모 `li`에 `position: relative;`를 부여하고 자식인 `.depth2`에 `position: absolute;`를 설정하여 1차 메뉴 바로 아래쪽에 정확히 배치

### 📰 국경없는의사회 소개 페이지
* **시맨틱 마크업 적용**: 콘텐츠의 성격에 따라 `<nav>`, `<figure>`, `<time>` 태그를 적재적소에 배치하여 구조의 명확성 확보
* **활성화 탭 스타일링**: 현재 선택된 메뉴 항목(`.about-menu .on a`)에 가상 요소(`::before`)를 사용하여 하단에 강조선(`height: 3px; background: #ea0519;`)을 동적으로 표현
* **멀티미디어 및 외부 데이터 연동**: 
  * `<iframe>` 태그를 활용해 외부 유튜브 플레이어를 반응형 크기(`width: 100%; height: 100%;`)로 임베드
  * 외부 이미지 경로 링크를 활용한 미디어 데이터 출력
* **카드 레이아웃 정렬**: 
  * 페이지 하단 '더 알아보기' 영역을 `display: flex;`와 `gap: 25px;`를 사용해 일정한 간격의 4열 카드 배열로 구현
  * 각 카드 내부의 카테고리 태그(`.cate`)를 `position: absolute;`를 사용해 이미지 좌측 상단에 겹쳐서 고정 배치

## 3. 코드 특징 및 웹 표준 규칙
* **텍스트 가독성 최적화**: 한글 단어가 중간에 끊겨서 줄바꿈되지 않도록 `.more-card .more-text p`에 `word-break: keep-all;` 속성 적극 활용
* **마크업 규칙 준수**: `<p>` 태그는 가장 작은 단위의 블록 요소이므로 내부에 `<div>`나 제목 태그(`<h1~6>`)를 중첩하지 않는 문법적 특성 준수 및 주석 기록
* **가상 요소를 활용한 UI 구현**: 의미 없는 빈 태그를 추가하는 대신 CSS 가상 요소(`::before`, `content: ''`)를 사용하여 레이아웃 디자인 요소 추가
