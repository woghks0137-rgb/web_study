# 📅 Day 3: HTML/CSS 기초 및 실습 (문화비축기지 따라 만들기)

오늘은 웹 페이지의 구역을 나누는 가장 기본적인 도구인 `div` 태그와, 디자인의 기본이 되는 선(Border) 및 폰트 설정 방법을 학습했습니다.

## 🚀 오늘 배운 내용

### 1. 구역 나누기 (`div`)
- 웹 페이지의 각 부분을 상자처럼 나누어 관리하는 `div` 태그의 용도와 사용법을 배웠습니다.
- 여러 개의 `div`를 사용하여 콘텐츠를 논리적으로 배치하는 연습을 했습니다.

### 2. 선과 테두리 설정 (`Border`)
- 요소의 테두리를 만드는 `border` 속성을 익혔습니다.
- **선의 두께**, **선의 모양**(실선, 점선 등), **선의 색상**을 각각 설정하는 방법을 실습했습니다.

### 3. 텍스트 및 간격 스타일링
- **폰트 설정**: 글꼴 모양과 텍스트의 크기, 줄 간격 등을 조절하여 가독성을 높였습니다.
- **요소 간격**: 선과 텍스트 사이, 혹은 구역 간의 간격을 조절하는 기초적인 방법을 배웠습니다.

## 💻 실습 프로젝트: 문화비축기지 클론
- 배운 내용을 바탕으로 '문화비축기지' 홈페이지의 텍스트와 이미지를 배치해 보았습니다.
- 이미지 파일(`img/`)을 경로에 맞춰 불러오고, HTML 구조에 맞게 화면을 구성했습니다.

## 💡 알게 된 점
- `div`를 어떻게 나누느냐에 따라 전체적인 페이지 구조가 달라진다는 것을 알게 되었습니다.
- 선 하나를 긋더라도 두께나 색상에 따라 페이지의 분위기가 많이 바뀐다는 점이 흥미로웠습니다.
