# 📚 웹 퍼블리싱 기초 학습 기록

## 📅 날짜
- 2026년 5월 9일

## ✍️ 학습 내용
오늘까지 진행한 실습을 통해 HTML 구조와 CSS 스타일링의 기초를 연습했습니다.

### 1. 스타벅스 홈페이지 따라하기 (Day 3)
- 웹 사이트의 기본적인 상단 바(Header)와 메인 레이아웃 구조를 잡는 법을 익혔습니다.
- 다양한 HTML 태그를 적재적소에 배치하는 연습을 했습니다.

### 2. 조선일보 기사 페이지 따라하기 (Day 4)
- 실제 뉴스 기사를 모델로 삼아 텍스트의 가독성과 이미지 배치 기술을 연습했습니다.
- `max-width`, `margin: 0 auto` 등을 활용해 본문을 중앙에 정렬하는 법을 배웠습니다.
- 줄 간격(`line-height`)과 배경색(`background-color`) 변경을 통해 기사처럼 보이게 하는 디테일을 학습했습니다.

## 💡 종합 소감
단순히 코드를 치는 것을 넘어, 실제 웹 페이지가 구성되는 원리를 직접 구현해보며 프론트엔드 개발자로서 필요한 가장 기본적인 실력을 쌓는 시간을 가졌습니다.
