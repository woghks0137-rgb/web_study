# 📝 HTML/CSS 기초 및 레이아웃 실습 기록

프론트엔드 개발자의 기본기인 **HTML의 의미론적(Semantic) 구조 설계**와 **CSS 박스 모델 및 레이아웃 정렬**을 학습한 프로젝트입니다.

## 🚀 실습 내용

### 1. 경주 여행 추천 가이드 페이지
* **목적**: 텍스트 정보를 구조화하고 가독성을 높이는 레이아웃 연습
* **핵심 학습 내용**:
    * 다양한 리스트 태그(`ul`, `ol`, `dl`)의 용도별 구분 사용
    * `mark`, `strong`, `em` 등 의미 중심의 텍스트 강조 태그 활용
    * `line-height`와 `margin` 조절을 통한 문서 가독성 개선

### 2. CSS 박스 모델(Box Model) 실습
* **목적**: 웹 요소의 크기와 여백을 결정하는 핵심 원리 이해
* **핵심 학습 내용**:
    * `width`, `height`, `border`, `padding`, `margin`의 관계 이해
    * `border-style(dashed 등)`과 `background` 속성 활용
    * 박스 내 여백과 외부 여백의 시각적 차이 학습

### 3. 쿠팡 뉴스룸 레이아웃 클론 코딩
* **목적**: 실제 서비스 중인 웹 페이지의 전문적인 레이아웃 재현
* **핵심 학습 내용**:
    * `max-width`와 `margin: 0 auto`를 활용한 중앙 정렬 시스템 구축
    * `dd` 태그의 기본 마진 초기화 및 `text-align: center`를 활용한 텍스트 배치

## 🛠 사용 기술
* **HTML5** (Semantic Markup)
* **CSS3** (Box Model, Layout, Typography)
* **Font**: Pretendard Variable (Web Font)
